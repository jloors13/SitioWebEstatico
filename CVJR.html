Proyectos Relevantes

Sistema para Boletería de Cine

Desarrollé una aplicación web para la gestión y venta de boletos de cine utilizando ASP.NET Core y React. La aplicación permite a los usuarios manejar la cartelera de películas, procesar ventas de boletos en tiempo real, gestionar ventas de bocadillos y combos, controlar el flujo de caja diaria, y generar informes de ventas e inventarios. Implementé autenticación y autorización para asegurar la integridad de los datos sensibles, como transacciones financieras.

Sistema para Cultivos

Creé una aplicación de escritorio en C# y C++ para la gestión de cultivos en fincas agrícolas. La aplicación incluye funcionalidades para registrar y monitorear etapas de cultivo, agregar, editar y eliminar registros de siembras, fertilizantes y cosechas, así como generar recordatorios para tareas de mantenimiento y categorización por tipo de cultivo. Se integró con bases de datos locales para un seguimiento eficiente y escalable.
